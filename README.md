# 타입스크립트

### 타입스크립트 설치
`` npm install -g typescript ``

### 자바스크립트 파일로 변환
`` tsc 파일명 ``

### 자바스크립트 파일로 컴파일 (watch mode)
`` tsc 파일명 -w``

### 컴파일러 툴 설치
`` npm install -g ts-node ``

## 실행
`` ts-node 파일명 ``
